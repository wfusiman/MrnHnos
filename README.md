# MnHnos
version 2 - Microsoft Visual Studio 2019 C# WPF.


Registro y control de servicios de mantenimiento para barcos:
- Armadores.
- Barcos.
- Balsas -> Valvulas hidrostaticas, Cilindros.
- Extintores.
- Cilindros.
- Sistemas fijos.
- Equipos autonomos.
- EEBD.
- Chalecos salvavidas.
- Trajes inmersion.
- VHF.
- SART.
- EPIRB.
- Reportes mantenimiento.
- Certificados mantenimiento.
